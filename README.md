# counter-app
This counter app could be my first project that is working really really good on desktop, ipad, tablets, and mobile it does look good on all of them.
despite it has fewer content but that content is made responsive through using css properties
for elements sizing for different screens, font size of buttons. 
also used flexbox here for majority of layout and alignment without hard coding the dimensions of elements.
also added few states for buttons and links so a user will know that they are interactive one.
